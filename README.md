# TastyBite - Food Ordering Website

A front-end food ordering website built with HTML, CSS, and JavaScript.

## Features

- Browse menu items by category (Burgers, Pizza, Drinks, Desserts)
- Add items to cart
- Adjust quantities with +/- buttons
- Remove items from cart
- Place orders via checkout form with validation
- Contact form for customer messages
- Fully responsive design for mobile and desktop

## How to Run

Open `index.html` in your browser.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
